## aniContents

※ HTML은 따로 추가하지 않음.


### ● js 설정
```javascript
aniCon = new aniContents(_page);
aniCon.init({
    voice: ['Voi4-3-2-01', 'Voi4-3-2-02', 'Voi4-3-2-03'],
});
```


### ● option

|옵션|설명|
|---|---|
|**voice**|음성(mp3) 파일을 배열 형식으로 사용합니다. (※ 확장자는 생략)<br>`voice: ['Voi4-3-2-01', 'Voi4-3-2-02', 'Voi4-3-2-03'],`|
